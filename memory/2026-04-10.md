## Key Learnings

### Multica + Claude Code 问题
- Claude Code 在 root 用户下无法非交互式运行
- 需要 `--bare` 参数跳过 CLAUDE.md
- multica daemon 会重启导致进程被 SIGTERM

### OpenClaw vs Claude Code
- 两者都能使用 agency-agents
- OpenClaw: 子任务方式调用，适合自动化
- Claude Code: 命令行 `--agent` 参数，适合交互

### GitHub 操作
- GitHub PAT 需要 `repo` scope 才能创建仓库
- SSH 方式推送比 HTTPS 更稳定
